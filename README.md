URL Shortner
============================

INSTALLATION
------------

#Git clone
```
git clone https://github.com/strelov1/url_shortner
cd url_shortner
```

#Composer install
```
composer global require "fxp/composer-asset-plugin:^1.2.0"
composer config -g github-oauth.github.com <gitub-token>
composer install
```

#Run docker
```
docker-compose up -d
```

#Browse url
```
http://localhost:8080
```
